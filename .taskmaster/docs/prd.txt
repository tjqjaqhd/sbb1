# 빗썸 데이트레이딩 자동매매 앱 PRD

## 프로젝트 개요
빗썸거래소 API를 활용한 실시간 데이트레이딩 자동매매 애플리케이션 개발

## 핵심 요구사항

### 1. 거래소 및 API 통합
- 빗썸 거래소 단독 사용
- 빗썸 API 2.0 버전 활용
- 실시간 시장 데이터 수집
- 계좌 정보 및 잔고 조회
- 실제 주문 실행 (매수/매도)
- API 키 관리 및 보안

### 2. 종목 선정 시스템
- 데이트레이딩에 적합한 종목 5개 자동 선정
- 거래량, 변동성, 유동성 분석
- 실시간 종목 순위 업데이트
- 종목별 기술적 지표 계산

### 3. 자동매매 엔진
- 실시간 시장 데이터 기반 매매 신호 생성
- 다중 매매 전략 구현
- 리스크 관리 시스템
- 손절매/익절매 자동 설정
- 포지션 크기 자동 계산

### 4. 실시간 모니터링
- 실시간 수익률 추적
- 매매 히스토리 로그
- 포트폴리오 현황 대시보드
- 알림 시스템

### 5. 백테스팅 시스템
- 과거 데이터 기반 전략 검증
- 성능 지표 분석
- 전략 최적화

### 6. 설정 관리
- API 키 설정 인터페이스
- 매매 전략 파라미터 조정
- 리스크 한도 설정
- 알림 설정

## 기술 스택
- Python (메인 언어)
- FastAPI (API 서버)
- WebSocket (실시간 데이터)
- SQLite/PostgreSQL (데이터베이스)
- React/Next.js (웹 인터페이스)
- Docker (배포)

## 보안 요구사항
- API 키 암호화 저장
- 2FA 인증 지원
- 로그 기록 및 감사
- 접근 권한 관리

## 성능 요구사항
- 실시간 데이터 처리 (< 100ms 지연)
- 주문 실행 속도 (< 500ms)
- 동시 다중 종목 처리
- 24/7 안정적 운영

## 데이터 요구사항
- 실제 시장 데이터만 사용 (목업 데이터 금지)
- 실시간 체결 데이터
- 과거 가격 데이터
- 거래량 및 호가 데이터

## 운영 요구사항

- 에러 처리 및 복구
- 로그 관리
- 성능 모니터링

## 사용자 인터페이스

- 설정 관리 패널
